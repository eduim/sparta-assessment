# Init

Install dependencies with

```bash
pnpm install
yarn
npm install
```

## Use it

Run the service with one of the following commands:

```bash
pnpm run dev
yarn dev
npm run dev

```

Go to `http://localhost:3000`

## Tests

Run the tests with one ofthe following commands:

```bash
pnpm run e2e
yarn run e2e
npm run e2e
```
